# MAPA - memory/

> O que vive nesta pasta: tudo que o agente precisa lembrar entre sessões.

## Estrutura

```text
memory/
├── MAPA.md
├── decisoes/
└── projects/
```

## Convenção de nomes

- `decisoes/{YYYY-MM}.md` - append-only
- `projects/{nome-curto}.md` - 1 por projeto, status atualizado

## Atualizar este mapa quando

- Adicionar uma nova subpasta
- Mudar a forma de registrar decisões
- Criar novos projetos

Cada pasta documenta a si mesma.
