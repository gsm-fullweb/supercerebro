# MAPA - skills/

> O que vive nesta pasta: skills modulares que o agente usa para tarefas recorrentes.

## Estrutura inicial

```text
skills/
├── MAPA.md
├── _registry.md
├── canais/
├── operacional/
├── planejamento/
└── starter/
```

## Como ler

- `starter/_registry.md` - jornada obrigatória
- `operacional/_registry.md` - utilitários e rotinas
- `planejamento/_registry.md` - skills de planejamento
- `canais/_registry.md` - canais adicionais

## Atualizar este mapa quando

- Instalar uma skill nova
- Criar uma categoria nova
- Mudar a organização dos registries

Cada pasta documenta a si mesma.
