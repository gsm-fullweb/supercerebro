# MAPA - archive/

> O que vive nesta pasta: versões antigas, arquivos substituídos e histórico manual.

## Regra

- Nada aqui é apagado sem motivo forte.
- Quando algo vira obsoleto, vai para archive.
- Use isso para preservar histórico sem poluir o workspace ativo.

## Convenção sugerida

- `archive/{nome}-{YYYY-MM-DD}.md`
- `archive/{nome}-{versao}.md`

## Atualizar este mapa quando

- O padrão de arquivamento mudar
- Surgir uma subpasta nova para histórico

Cada pasta documenta a si mesma.
