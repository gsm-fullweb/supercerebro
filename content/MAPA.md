# MAPA - content/

> O que vive nesta pasta: tudo que o agente cria para o Richard (posts, drafts, imagens, scripts).

## Estrutura

```text
content/
├── MAPA.md
├── drafts/
└── archive/
```

## Convenção de nomes

- `drafts/post-{tema-curto}-{YYYY-MM-DD}.md`
- `drafts/email-{destinatario}-{YYYY-MM-DD}.md`
- `drafts/video-{titulo-curto}-{YYYY-MM-DD}.md`

Quando algo é publicado, mover de `drafts/` para `archive/`. Não deletar.

## Atualizar este mapa quando

- Adicionar nova subpasta
- Mudar convenção de nomes

Cada pasta documenta a si mesma.
