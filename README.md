# SuperCérebro v2 — Fullweb AI-Native Agency

Este repositório é a fonte única da verdade para os agentes da Fullweb.

---

## Estrutura

```
supercerebro/
├── AGENTS.md                    ← Quem são os agentes e como se comunicam
├── memory/
│   ├── businesses.md            ← Empresas do grupo e receitas
│   ├── seo-workflow.md          ← Pipeline SEO Programático detalhado
│   └── clients.md               ← Clientes ativos (criar conforme necessário)
├── projects/
│   ├── fullweb.md               ← Projeto agência AI-Native
│   ├── chathook.md              ← Projeto SaaS ChatHook
│   ├── zapcode.md               ← Projeto ZapCode
│   └── careconnect.md           ← Projeto CareConnect
├── logs/
│   ├── diario.md                ← Log diário de decisões e ações
│   └── publicacoes.md           ← Log de conteúdos publicados
└── README.md                    ← Este arquivo
```

---

## Hierarquia de Leitura

Qualquer agente, antes de agir, deve ler nesta ordem:

1. `AGENTS.md` — entender seu papel e as regras de comunicação
2. `memory/businesses.md` — entender o ecossistema de negócios
3. Arquivo de projeto relevante em `projects/`
4. `logs/diario.md` — ver o que foi feito recentemente
